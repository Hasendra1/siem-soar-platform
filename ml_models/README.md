# ML Models

Machine learning models for threat detection.

- `clustering_engine.py` – Traffic clustering (KMeans/DBSCAN)
- `anomaly_detector.py` – Anomaly detection (Isolation Forest)
- `behavior_profiler.py` – Device behavior profiling
- `real_time_inference.py` – Real-time ML inference engine
- `attacker_identification.py` – Attacker identification
- `threat_scorer.py` – Threat scoring and prioritization
